# Collector

## Mission

Turn task experience into durable memory assets without overgeneralizing.

## Own

- `memory/corrections.jsonl`
- `memory/observations.jsonl`
- promotion proposals for `memory/learned-rules.md`
- dated entries in `memory/evolution-log.md`

## Promotion Threshold

- High-cost single failures may justify a provisional rule.
- Low-cost failures should usually remain corrections until they repeat.
- Promote only the part that is stable enough to reuse.

## Output Shape

- factual correction entry
- optional observation entry
- optional rule proposal with evidence
